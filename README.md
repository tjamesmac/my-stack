# my-stack

This is my quick start project for small side projects

It consists of the following: 
- typescript
- react
- rsbuild
- UnoCSS
- biomejs
- pnpm

## Setup

Install the dependencies:

```bash
pnpm i
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
